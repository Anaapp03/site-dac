# 🤖 Automação de Processos em Software

Este é um site institucional/informativo desenvolvido para apresentar conceitos, benefícios e práticas sobre a **Automação de Processos em Software**. O projeto foi criado com o objetivo de demonstrar como a tecnologia pode otimizar o fluxo de trabalho no desenvolvimento de sistemas, reduzir erros humanos e acelerar entregas.

---

## 🚀 Funcionalidades do Site
* **Página Inicial Dinâmica:** Apresentação clara sobre o impacto da automação no mercado atual.
* **Seções Informativas:** Explicações sobre CI/CD (Integração e Entrega Contínuas), testes automatizados e monitoramento.
* **Design Responsivo:** Adaptado para leitura tanto em computadores quanto em dispositivos móveis.
* **Interface Moderna:** Foco na experiência do usuário com navegação fluida.

---

## 🛠️ Tecnologias Utilizadas
O projeto foi desenvolvido utilizando as tecnologias web fundamentais:
* **HTML5:** Estruturação semântica de todo o conteúdo do site.
* **CSS3:** Estilização, layout, cores e responsividade.
* **JavaScript (opcional):** Interatividade e dinamismo na página.

## 🚀 Como Executar o Projeto Localmente

Se você deseja visualizar ou editar este site no seu computador, siga os passos abaixo:

1. **Baixar o projeto:** Clique no botão verde **"Code"** (no topo desta página) e selecione **"Download ZIP"**. Após o download, extraia os arquivos no seu computador.
2. **Abrir no VS Code:** * Abra o aplicativo **Visual Studio Code**.
   * Vá em `Arquivo` > `Abrir Pasta` (ou `File` > `Open Folder`) e selecione a pasta extraída do projeto.
3. **Executar o site:** Dentro do VS Code, localize o arquivo `index.html`, clique com o botão direito nele e escolha a opção para abrir no seu navegador (ou utilize a extensão *Live Server*).
